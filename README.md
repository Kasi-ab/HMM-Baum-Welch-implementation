# Hidden Markov Model using Baum-Welch Algorithm

Name: KASINATHAN A B

University Registration Number: TCR24CS042
## Description
This project implements a Hidden Markov Model (HMM) using the Baum-Welch algorithm (Expectation-Maximization) for parameter estimation.

The model includes:
- Forward Algorithm
- Backward Algorithm
- Gamma and Xi computation
- Parameter re-estimation

A Streamlit web app is provided for visual interaction.

## How to Run

Install dependencies:

pip install -r requirements.txt

Run app:

streamlit run app.py

## Files

- hmm.py → HMM and Baum-Welch implementation
- app.py → Streamlit visualization
- README.md → Documentation
